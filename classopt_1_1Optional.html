Detailed Description

template<typename T>
class opt::Optional< T >

Wraps a type to provide an optional 'null', or empty state.

A wrapped object is accessed by dereferencing an Optional object. Optional objects can be tested as a boolean for whether or not they contain a value. Useful when 0, -1, or default constructed does not suffice for 'no value'.

Typical usage:

Optional<int> opt_i{5};
if(opt_i) {
foo(*opt_i);
}
