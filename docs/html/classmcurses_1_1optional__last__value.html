Detailed Description

template<typename T>
class mcurses::optional_last_value< T >

A functor class that returns the last value in an iterator range.

Attempts to dereference every iterator in the range [first, last). If the iterator range is empty, an empty Optional object is returned, otherwise an Optional wrapping the last value is returned.
